# Differential expression and functional analysis of yeast transcriptomics data


## **1 | Introduction**
The advent of RNA sequencing (RNA-Seq) has resulted in a surge of high-resolution gene expression data (Marioni et al., 2008), with this data covering a much greater part of the transcriptome compared to data generated by other methods like Sanger sequencing or microarray methods (Kukurba & Montgomery, 2015). Consequently, the technique of differential gene expression (DGE) analysis, which determines if and what genes are differentially expressed between one group and another, has arisen (Rosati et al., 2024). RNA-Seq, in conjunction with DGE analysis and other methods, can thus identify if genes are involved in a biological process or experimental condition, allowing for important insights to be discovered. Notably, RNA-Seq has led to the discovery of previously unknown gene structures, of diverse splice variants, and of the biased expression of specifc alleles (Kukurba & Montgomery, 2015), and has elucidated details on the up- or downregulation of genes. 


The negative binomial distribution is typically used to model RNA-Seq data as it accounts for the biological variation between replicates. As such, parametric methods such as edgeR and DESeq2 are commonly used to analyze RNA-Seq data in order to ensure statistical accuracy (Rosati et al., 2024). Contrasting the two methods, DESeq2 is more conservative than edgeR and generally has better control for the false discovery rate (Soneson & Delorenzi, 2013). Moreover, edgeR showed poorer control of the false positive rate when tested with yeast transcriptome data (Nookaew et al., 2012). Additionally, DESeq2 is more typically employed for data obtained from controlled experimental conditions, while edgeR shows better performance with more complex and highly variable data (Rosati et al., 2024).

## **2 | Methods**

### **2.1 | Data Description**
The transcripts for the yeast dataset were obtained from the [BioProject page (Accession: PRJNA592304)](https://www.ncbi.nlm.nih.gov/bioproject/PRJNA592304), which was generated from the study conducted by Mardanov et al. (2020). In total, nine SRA files were downloaded and coverted to FASTQ format using SRA Toolkit (v3.2.1). The reference genome, transcriptome, and GTF annotation file for _Saccharomyces cerevisiae (strain S288C)_ was obtained from the NCBI genome database. These files were renamed to ref.fna, rna.fna, and genomic.gtf, respectively.

```
prefetch SRR105516{57..65}
fasterq-dump SRR105516{57..65}

datasets download genome accession GCF_000146045.2 --include genome,rna,gtf --filename yeast_ref.zip
```

### **2.2 | Quality Control**
FastQC (v0.12.1) was used to inspect the quality of the reads pre-quantification. The resulting fastqc.html files were combined into a single report using MultiQC (v1.33). According to the summary figures generated by MultiQC, the reads had high per base quality scores, with most exceeding a Phred score of 35 (Fig. 1). Although the summary heatmap reported unusual results for per base sequence content and sequence duplication levels (Fig. 2), these are expected for RNASeq reads, as they reflect biases or artifacts in library preparation (Shi et al., 2021). Moreover, trimming reads appears to have no discernible effect on the accuracy of quantification (Liao & Shi, 2020). As such, no trimming or pre-filtering was performed pre-quantification.
```
fastqc *.fastq -o ./fastqc_results
cd fastqc_results/
multiqc ./
```

### **2.3 | Quantification **
Quantification of the transcripts was performed using Salmon (v1.10.3). A full selective alignment paradigm, as described by Srivastava et al. (2020), was used during quantification. This involved creating a decoy file with the reference genome as the decoy sequences. An index was created using the concatenated transcriptome and genome, along with the decoy sequences.
```
grep "^>" ref/ref.fna | cut -d " " -f 1 > decoys.txt
sed -i 's/>//g' decoys.txt
cat ref/rna.fna ref/ref.fna > genome_transcripts.fa
salmon index -t genome_transcripts.fa -d decoys.txt -p 6 -i transcripts_index
```
Salmon was then used to quantify counts for each read. The script used to iterate over the reads is described in [quantify_reads.sh](scripts/quantify_reads.sh). The flags --gcBias and --seqBias were employed to mitigate the effects of potential GC bias and sequence bias in the reads.
```
salmon quant -i transcripts_index -l A -r ../reads/${read}.fastq -p 6 --validateMappings --gcBias --seqBias -o quants/${read}
```


### **2.4 | Differential Gene Expression Analysis and Functional Annotation**
Differential expression analysis and functional enrichment analysis were conducted on the quantified counts generated by Salmon. The full pipeline and code used for these analyses and the resulting figures is described in [yeast_transcriptome_analysis.R](scripts/yeast_transcriptome_analysis.R). The DESeq2 package (v1.50.2) was used to generate differential gene expression results. The design formula included stage (i.e. early, thin, mature) as the factor. Log fold change shrinkage was then performed on the resulting DESeqDataSet objects. Pairwise comparisons were performed (e.g. thin vs early, mature vs early, mature vs thin).
```
dds_TXI <- DESeqDataSetFromTximport(txi,
  colData = df_yeast_metadata,
  design = ~stage
)
dds_e <- DESeq(dds_TXI)

evt_LFC <- lfcShrink(dds_e,
  coef = "stage_thin_vs_early",
  type = "apeglm"
)
```
Functional annotation and over-representation analysis was conducted using the clusterProfiler package (v 4.18.4). Both GO and KEGG enrichment were performed to determine any overrepresented biological processes or metabolic pathways in each stage.

A PCA plot and a heatmap were created to visualize the overall structure of the transcriptomic data post-quantification and pre-analysis. As seen in the PCA plot, the replicates cluster well together, and the experimental groups (i.e. stages) are separate from each other (Fig. 3). PC1 explains 72% of the variance in the data, with the groups showing no overlap along this axis. PC2 explains 24% of the variation, with the thin stage separated from the early and mature stages along the y-axis. The heatmap shows the same pattern seen in the PCA plot, with the samples clustering together by stage (Fig. 4). Sequences from each stage are distinct from each other, as represented by the darker orange and red colors. Sequences in the early and mature samples are similar among themselves, while sequences in the thin samples show slightly more variability.

### **3.2 | Differential Gene Expression**
Two MA plots for each pairwise comparison were created, with top genes being labeled according to either the adjusted P-value or the log fold change. Colored points were filtered as statistically significant (p < 0.05), and further divided into upregulated (red) and blue (downregulated) with a cutoff log fold change value of 2. In the thin vs early comparison, there were 485 upregulated genes and 448 downregulated genes, for a total of 933 significant genes. In the mature vs thin comparison, there were 1086 significant genes (627 upregulated and 459 downregulated), while in the mature vs early comparison, there were 1715 significant genes (951 upregulated and 764 downregulated). Although each comparison had the same number of samples, the early vs mature comparison appears to have a greater proportion of statistically significant genes, as seen in Figure 6. Certain genes appear to be consistently downregulated in specific stages, such as YHR094C, YGR087C, and YGR296W in the thin and mature stages (Fig. 5, Fig. 6). ## **4 | Discussion**

### **4.1 | Overall Data Structure**
The transcriptomic data obtained from the study appear to be of good quality, as replicates cluster according to stage (Fig. 3, Fig 4). This clustering suggests that each experimental group has a distinct transcriptomic profile, with different collections of genes expressed during each stage of velum formation. PC1, now identified as the stage, explains 72% of the variation in the dataset (Fig. 3). Replicates in the early stage and mature stage are greatly separated along this component, reflecting the much greater variablity in gene expression levels between the early and mature stages. Notably, the PCA plot shows a V-shaped trend, with the thin stage separated from the early and mature stages. PC2 may then reflect a transcriptomic shift as yeast shifts away from glycolysis and moves towards oxidative metabolism (Gutierrez, Roldan, & Pérez, 2010), thus expressing a set of gluconeogenic or stress-related genes distinct from those in the early and mature stages. This is supported by the heatmap (Fig. 4), which shows that replicates in the thin stage show greater variability within themselves. In comparison, the early and mature replicates demonstrate greater in-group similarity. It is key to note that overall, each stage's replicates are distinct and well-separated from replicates of other stages.

### **4.2 | Biological Implications**
Differential gene expression and functional enrichment analyses of the yeast transcripts then provide interesting biological insights as to the expression of genes during each stage of velum formation. The stages of velum formation are marked by changes in environmental conditions, which consequently place pressure on the yeast to respond. As such, changes in the expression of genes reflect yeast's adaptations.
